# Folder Media Impor

Letakkan gambar atau audio di folder ini ketika membuat paket ZIP impor.

Aturan gambar:

- PNG, JPEG, atau WebP.
- Maksimum 100 KB per file.
- Nama file harus sama persis dengan nilai `image_file` atau `question_image` pada Excel.
- Hindari spasi ganda dan karakter path seperti `../`.

File `sample-reading-chart.webp` adalah contoh berukuran kecil.
