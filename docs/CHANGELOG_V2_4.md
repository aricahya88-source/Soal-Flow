# Changelog v2.4.0

- Logo aplikasi dan favicon baru berbasis SVG.
- Plotting validator diubah dari pemilihan soal menjadi pemilihan kode kisi-kisi.
- Plotting tugas dipisah menjadi submenu penulis soal dan validator soal.
- Setiap kisi-kisi otomatis membuat slot serta kode soal sesuai target jumlah soal.
- Menu Tulis Soal menampilkan daftar soal di bagian atas dan dikelompokkan per kisi-kisi.
- Penulis mengisi slot melalui tombol Buat; kode soal tidak perlu dibuat manual.
- Daftar kisi-kisi dipindahkan ke atas, form tambah bersifat collapsible, dan kolom tabel dapat di-hide/unhide.
- Dashboard didesain ulang dan panel Alur v2.2 dihapus.
- Data plotting validator lama dinormalisasi otomatis menjadi cakupan seluruh slot dalam kisi-kisi yang sama.
