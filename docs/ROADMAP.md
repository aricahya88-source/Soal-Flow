# Roadmap

## Fase 1 — Fondasi

- Monorepo
- Tema dan aksesibilitas
- Auth/RBAC
- Database dan audit
- Storage

## Fase 2 — Authoring

- Kisi-kisi dan versi
- Stimulus
- Soal pilihan ganda

- Validator bertingkat

## Fase 3 — Impor dan laporan

- Excel + ZIP media
- Preview dan staging
- PDF kisi-kisi/kartu soal/validasi
- Word terkontrol

## Fase 4 — Delivery

- Paket ujian
- Snapshot
- Randomisasi blok stimulus
- Jadwal

## Fase 5 — Exam

- Login/token
- Timer server
- Autosave
- Resume
- Submit
- Scoring tanpa penalti

## Fase 6 — Hardening

- Load test
- Monitoring
- Backup/restore drill
- Rate limiting
- Security review
- Analitik kualitas soal
